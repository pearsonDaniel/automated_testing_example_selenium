# MTT Selenium Test Suite
## For the DODIAC MIPR Tracking Tool

## Installations and Setup

This test suite requires python, selenium, and other dependencies

After cloning the project and installing Python, run these commands to install its dependencies.



```sh
python get-pip.py
pip install pip install -U selenium
pip install pytest
pip install pytest-html
pip install requests
pip install openpyxl
pip install faker
```
If pip needs to be updated then run:
```sh
python.exe -m pip install --upgrade pip
```


Check versions...

```sh
pip freeze
```
